# Regex Data Extraction Project

This project demonstrates how to use **Python Regular Expressions** to extract useful information from raw API responses.

## Extracted Data
The script can detect:
- Email addresses
- URLs
- Phone numbers
- Credit card numbers
- Time (12-hour and 24-hour)
- HTML tags
- Hashtags
- Currency amounts
